    Définitions d'un tableau :

    Les tableaux sont des variables permettant de stocker plusieurs éléments en même temps comme des nombres, des chaînes de caractères, etc.
    Les tableaux stockent le plus souvent un seul type d'éléments, mais les langages comme JavaScript et Python autorisent le mélange de plusieurs types dans un même tableau.
